# Compile Vifm under Ubuntu

Get sources from here:
http://vifm.info/downloads.shtml

Prerequisites: 

    $ sudo apt-get install libncursesw5-dev

Build:

    $ ./configure
    $ make
    $ sudo make install

Vifm will be installed into your "/usr/local/bin" folder. 

Uninstall:

    $ sudo make uninstall
